条件永远为true或false的if语句是没必要村爱的，也让降低代码的可读性。

如下代码所示：

public void myMethod() {
  if (true) {                   // Non-Compliant
    doSomething();
  }
}

应该重构为：

public void myMethod() {
  doSomething();                // Compliant
}

如下代码所示：

public void myMethod() {
  if (false) {                  // Non-Compliant
    doSomething();
  }
}

应该重构为：

public void myMethod() {        // Compliant
}
