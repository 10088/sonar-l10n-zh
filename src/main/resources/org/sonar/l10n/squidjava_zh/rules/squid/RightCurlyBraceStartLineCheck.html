分享一些代码规则是让团队高效协作的一个关键因素。 这个规则要求右大括号比如位于代码行的前端。

如下代码演示了这个规则： 

public void myMethod() {
  if(something) {
    executeTask();}                   // Non-Compliant
  else if (somethingElse) {
    doSomethingElse();
  }                                   // Compliant

  try {
    generateOrder();
  } finally {                         // Compliant
    closeConnection();
  }                                   // Compliant

  try {
    generateOrder();
  } finally {                         // Compliant
    closeConnection(); }              // Non-Compliant
}
