分享一些代码规则是让团队高效协作的一个关键因素。 这个规则要求左大括号比如位于代码行的末端。

下面代码演示了这个规范： 

public void myMethod() {              // Compliant
  if(something)
  {                                   // Non-Compliant
    executeTask();
  } else {                            // Compliant
    doSomethingElse();
  }
  if( param1 && param2 && param3
    && something3 && something4)
  {                                   // Non-Compliant
    executeAnotherTask();
  }
}
