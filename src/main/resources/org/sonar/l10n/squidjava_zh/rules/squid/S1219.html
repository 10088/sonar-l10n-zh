即使是合法的，把条件和无条件标签同时使用在switch语句中也是非常令人迷惑的，并可能误导开发者。

不符合规则的代码示例

switch (variable) {
  case 0:
    /* ... */
  myLabel:           // Noncompliant
    result = 0;
    break;
  case 1:
    /* ... */
  default:
    /* ... */
}

符合规则的方案

switch (variable) {
  case 0:
    /* ... */
    result = 0;
    break;
  case 1:
    /* ... */
  default:
    /* ... */
}
