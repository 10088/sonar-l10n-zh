字符串文本，和其他Object一样，应该使用equals()方法进行比较。 ==和!=通常不会起作用。

如下代码：

if (variable == "foo") { /* ... */ }         // Non-Compliant
if (variable != "foo") { /* ... */ }         // Non-Compliant

应该重构为：

if ("foo".equals(variable)) { /* ... */ }    // Compliant
if (!"foo".equals(variable)) { /* ... */ }   // Compliant
